# Algoritmia - Guía de Estudio

## Sobre este repositorio

Este repositorio está organizado para ser recorrido **más de una vez**.

Muchos temas, incluso los que aparecen en las primeras secciones, tienen capas de profundidad que solo se comprenden completamente después de haber visto algoritmos más avanzados.

## Cómo usar este repositorio

### Primera vuelta

- Recorre los temas en orden numérico
- Enfócate en entender los conceptos principales
- Implementa los ejemplos básicos
- No te detengas demasiado si algo no queda 100% claro

### Segunda vuelta

Después de haber completado temas como Sorting, Graphs y Dynamic Programming:

- Vuelve a las primeras secciones
- Verás conexiones que antes no notabas
- Entenderás por qué ciertas técnicas funcionan
- Las complejidades y trade-offs cobrarán sentido
